# Модуль фитнес-трекера


Пакет spentenergy:
Функции для расчёта потраченных калорий при ходьбе, беге, а также функции для расчёта средней скорости и дистанции.

Пакет personaldata:
Структура с данными пользователя.

Пакет trainings:
Функционал по разбору строки с данными о тренировках и формирования строки с информацией о них. 

Пакет daysteps:
Функционал для парсинга строки с данными о прогулках и формирования строки с информацией о них. 

Пакет actioninfo:
Реализует вывод общей информации обо всех тренировках и прогулках.
